# 概要

自分用適当記録表。
数週間〜数ヶ月に一回程度発生するタスクの備忘録。Google Spreadsheetとの連携版。

やること：
1. ホーム画面でやったタスクを登録する。
    *  「今日」「昨日」＋任意の日付を入力
2. 設定画面でタスクを編集して、実行間隔（周期）を設定する。（ここでタスク追加も可能）

やられること：
1. 最終実行日から期限超過しているタイプのものがホーム画面に表示される。


# 準備

適当にスプレッドシートを作成。

* items シート：1行目の1列目から5列目までに、id	name	period	color	lastDate の5つの名前を順に記入。
    * タスク項目が追加される。
* logs シート：1行目の1列目から3列目までに、timestamp	date	name の3つの文字列を順に記入。
    * （部分的にログが追加される）

1行目を固定しておくと見やすい。


Google App Scriptで、
「ファイル」に以下を配置

* Code.gs（元の中身を置き換える）
* index.html（このファイルの名前を変えた場合は `Code.gs` の該当箇所を変更する）

「デプロイ」：種類を選択するとき「ウェブアプリ」を選択。デプロイ後に出現したURLにアクセスすると使えるはず。
※タイムラグがある。
